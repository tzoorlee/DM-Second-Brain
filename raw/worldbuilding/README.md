# Worldbuilding — Design Material

> Drop material here to build your world BEFORE playing:
> draft cities, factions, NPCs, plot hooks, lore.
>
> The agent processes it with the DESIGN INGEST operation: it creates wiki entities
> with `canon_status: prepared`. When the PCs touch them in play, they become `established`.
>
> No order needed: write freely, the agent structures it.
