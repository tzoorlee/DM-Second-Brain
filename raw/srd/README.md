# SRD 5.1 — System Material

> Place the SRD 5.1 (System Reference Document) for D&D 5e here.
> Released under Creative Commons Attribution 4.0 (CC-BY-4.0).
> This is SYSTEM material, distinct from your custom world.

## What goes here
- SRD monster statblocks
- SRD spells
- SRD rules, conditions, magic items

## How the agent treats it
- Canon as a SYSTEM RULE, not as your world's lore.
- When creating a custom monster or NPC, the agent may base it on an SRD
  statblock ("use the Veteran statblock for this guard").
- Distinct from world-bible.md (YOUR world) and house-rules.md (YOUR variants).
